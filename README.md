# ReciclABC

A rede que conecta condomínios e compradores de recicláveis no Grande ABC Paulista.

## O que é

O ReciclABC é uma plataforma de intermediação entre geradores (condomínios residenciais e comerciais) e compradores (recicladores, cooperativas, empresas) de materiais recicláveis — vidro, plástico, papel e metal — no Grande ABC Paulista.

Diferenciais:
- Preço de referência calculado pelo mercado local (caminho rápido)
- Oferta personalizada opcional para quem quer negociar
- Coleta agregada por rotas otimizadas (mais material por viagem)
- Relatório ESG automático para os condomínios
- Pagamento via PIX com split automático
- Reputação bidirecional para proteger as duas pontas

## Estágio

Versão beta / MVP. Site estático para validação de mercado e captação dos primeiros clientes.

Entrada de demanda real acontece via WhatsApp Business (fora deste site, por enquanto).

## Como rodar localmente

Basta abrir o arquivo `index.html` em qualquer navegador. Não há build, nem dependências externas instaladas localmente — fontes e ícones são carregados via CDN.

## Como publicar no ar

Este repositório está configurado para GitHub Pages. Veja o passo a passo abaixo na descrição do projeto.

- Fontes: Inter (Google Fonts)
- Ícones: Tabler Icons (via CDN)
